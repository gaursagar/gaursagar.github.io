</p>

<p>
To run the examples locally, clone this repo, install <a href="http://nodejs.org/" target="_blank">node.js</a>, and, from the `cesium-google-earth-examples` root directory, run
<pre>
npm install
node server.js
</pre>
Browse to 
<pre>
http://localhost:8000/
</pre>
