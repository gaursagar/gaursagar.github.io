About

I have made this app using CesiumJS. It allows you to quickly visualize data (about US states) as 3D plots on the map.

You can fill in the data using the forms
(TIP: If you are lazy to fill the data press the Generate Random Data button to fill the columns with random numbers.)

Press button Go to plot the data.
